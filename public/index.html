API Developer Sandbox

Welcome to the developer playground for Celestia Restaurant API server. All requests are routed through simulated latency to verify loading states on the client frontend.

Base URL: http://localhost:3000

Menu Catalog Endpoints

GET /api/menu

Retrieves the complete list of dishes. Supports optional filters: ?category=burgers or search keywords: ?search=truffle.

GET /api/menu/:id

Fetches details of a single dish, incrementing its popularity counters.

Reservations Data Persistence

GET /api/reservations

Fetch all submitted customer reservation requests. Filter by status: ?status=Pending or search query string.

POST /api/reservations

Submits a new table reservation inquiry record into JSON DB.

Order Checkout Simulation

POST /api/checkout

Places a new combined order. Re-validates prices server-side to prevent fraud.

GET /api/orders

Fetch details of all completed checkout transactions.

Dashboard Analytics Stats

GET /api/stats

Fetch aggregated stats values to populate dashboard statistics cards.
